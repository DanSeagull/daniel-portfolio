# Test Plan: E-commerce Website | Тест-план: Интернет-магазин

## 1. Scope | Объём тестирования
[EN] Testing the main user journey: login → search → add to cart → checkout
[RU] Тестирование основного пути пользователя: логин → поиск → добавление в корзину → оформление заказа

## 2. Test Types | Типы тестирования
- Functional testing | Функциональное тестирование
- UI testing | Тестирование интерфейса
- Regression testing | Регрессионное тестирование

## 3. Tools | Инструменты
- JIRA (bug tracking)
- TestRail (test case management)
- Postman (API testing)

## 4. Entry Criteria | Критерии входа
- Test environment is ready
- Test data is prepared

## 5. Exit Criteria | Критерии выхода
- 95% of test cases passed
- No critical/blocker bugs remain
