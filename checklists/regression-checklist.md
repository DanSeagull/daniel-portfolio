# Регрессионный чек-лист: Яндекс Go (перед релизом)

## 1. Авторизация и профиль
- [ ] Вход по номеру телефона (SMS-код)
- [ ] Вход с невалидным номером → ошибка
- [ ] Смена способа оплаты (карта → наличные)
- [ ] Просмотр истории поездок (последние 5 поездок отображаются)

## 2. Вызов такси (основной сценарий)
- [ ] Автоопределение текущего адреса (погрешность < 50 м)
- [ ] Ручной ввод адреса "Откуда" и "Куда"
- [ ] Выбор тарифа (Эконом, Комфорт, Бизнес, Детский)
- [ ] Кнопка "Заказать" активна только при заполненных полях
- [ ] Появление карточки водителя после подтверждения

## 3. Платежи
- [ ] Списание средств с привязанной карты
- [ ] Оплата наличными (нет предварительного холдирования)
- [ ] Промокод: применение, отображение скидки
- [ ] Промокод: просроченный → ошибка

## 4. Алиса (AI-чат)
- [ ] Голосовой ввод "Поехали домой" → строится маршрут до домашнего адреса
- [ ] Голосовой ввод с шумом (эмуляция) → хотя бы 70% точность
- [ ] Текстовый ввод "Привези меня на вокзал" → предлагает список вокзалов

## 5. Работа в фоне и сеть
- [ ] Приложение не вылетает при сворачивании во время поездки
- [ ] Потеря интернета на 10 секунд → поездка не отменяется
- [ ] Восстановление интернета → статус поездки обновляется
