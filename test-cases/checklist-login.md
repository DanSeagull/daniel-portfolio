# Login Page Test Checklist | Чек-лист тестирования страницы логина

## Positive Tests | Позитивные тесты
- [ ] Login with valid email and password
- [ ] "Remember me" checkbox saves session
- [ ] Password reset link works

## Negative Tests | Негативные тесты
- [ ] Login with invalid email shows error
- [ ] Login with invalid password shows error
- [ ] Empty fields show validation errors
- [ ] Too many failed attempts → account locked

## UI Tests | UI-тесты
- [ ] All elements are visible and aligned
- [ ] Password field masks input with dots
- [ ] Buttons are clickable
