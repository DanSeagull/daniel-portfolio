# Баг-репорт: Некорректное определение адреса при слабом сигнале GPS

**Заголовок:** GPS определяет адрес с ошибкой > 500 метров в районе высотных зданий

**Серьезность:** Major  
**Приоритет:** High

**Окружение:**
- Устройство: iPhone 15, iOS 17.2
- Приложение: Яндекс Go 4.2.1
- Локация: Москва, деловой центр "Москва-Сити" (башня Федерация)

**Шаги воспроизведения:**
1. Открыть приложение Яндекс Go
2. Находиться в высотном здании (25+ этаж/75+ м)
3. Нажать на поле "Откуда" (автоопределение)
4. Посмотреть предложенный адрес

**Фактический результат:**  
Приложение определяет адрес как "Москва, набережная Тараса Шевченко, 23" (соседнее здание). Ошибка ~600 метров.

**Ожидаемый результат:**  
Должен определяться корректный адрес "Москва, Пресненская набережная, 12" или предложить ввести вручную.

**Скриншот:**

**Дополнительная информация:**  
Проблема воспроизводится только в режиме экономии заряда (Low Power Mode). При отключении режима экономии — GPS работает корректно.
