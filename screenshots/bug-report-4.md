**Заголовок:** API возвращает 500 Internal Server Error при отправке эмодзи 🚗 в комментарии к заказу

**Серьезность:** Minor  
**Приоритет:** Low

**Окружение:**
- API endpoint: POST /api/orders/create
- Инструмент: Postman

**Шаги воспроизведения:**
1. Отправить POST запрос на создание заказа
2. В поле "comment" добавить текст с эмодзи: "Жду у подъезда 🚗"
3. Отправить запрос

**Фактический результат:**  
HTTP 500 Internal Server Error

**Ожидаемый результат:**  
HTTP 201 Created, эмодзи корректно сохраняется или заменяется на текст

**Запрос:**
```json
{
  "pickup": "Тверская 1",
  "dropoff": "Кремль",
  "comment": "Жду у подъезда 🚗"
}
